# FRUTTÉ — Pulpa de Fruta Pasteurizada (Colombia)

Estudio de viabilidad, modelo financiero y website comercial para FRUTTÉ — negocio de pulpa de fruta pasteurizada de larga vida (sin cadena de frío) en Colombia.

**Socios:** Luigy Roso & Fabián (SAS 50/50)

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `viabilidad-financiera.html` | Dashboard financiero interactivo — escenarios A+ y B, P&L, proyecciones 10 años, distribución por socio |
| `estudio-mercado.html` | Estudio de mercado completo — mercado, competencia, regulación, equipos, distribución, exportación, DOFA |
| `website-comercial.html` | Website comercial de la marca FRUTTÉ (B2B + B2C) |
| `investigacion-mercado.md` | Datos crudos de investigación (fuentes, precios, cifras) |
| `regulacion-impuestos-colombia.md` | Normativa colombiana: SAS, impuestos (RST/IVA/ICA), seguridad social, INVIMA |
