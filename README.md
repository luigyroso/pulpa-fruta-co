# Pulpa de Fruta Pasteurizada — Estudio de Mercado Colombia

Estudio de viabilidad de mercado para un negocio de pulpa de fruta pasteurizada (larga vida, sin cadena de frío) en Colombia.

**Live:** [https://luigyroso.github.io/pulpa-fruta-co/](https://luigyroso.github.io/pulpa-fruta-co/)

- Tesis central: pasteurizada vs. congelada
- Tamaño del mercado y tendencias
- Competencia directa e indirecta
- Proceso de pasteurización y equipos
- Materia prima y precios
- Canales de distribución
- Oportunidad de exportación
- Regulación INVIMA
- Análisis financiero y ROI
- Análisis DOFA
- Hoja de ruta
